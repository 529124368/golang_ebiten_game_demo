# 通过golang的ebiten引擎做的类传奇小demo  power by ebiten引擎

！转载请注明来源

```
下载源码
1. go mod vendor 
2. go run main.go   运行游戏
```
## 控制说明
  1.目前demo 支持 上 下 左 右 按键移动 和 鼠标右键点击移动。
  
### 更新履历
  1. 左上角信息提示 更新！！
